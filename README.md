# sistema-registro-venda-fruta
Sistema de venda e registro de frutas 

Este é um projeto desenvolvido para gerenciar o **estoque** e **venda** de uma loja de frutas.

##Funcionalidades:
- **Cadastro de Frutas:** Registro de nome, quantidade e preço.
- **Registro Temporal:** Uso da biblioteca `datetime` para marcar o momento exato da entrada no estoque.
- **Listagem de Itens:** Visualização clara de todos os produtos cadastrados e seus respectivos horários.
- **Remoção:** Exclusão de itens do estoque.

##Tecnologias Utilizadas:
- **Linguagem:** Python 3
- **Biblioteca Interna:** `datetime` (Standard Library)
- **Estrutura de Dados:** Listas e Dicionários
